# P.I.E.

**Permanently Incomplete Environment**

P.I.E. is a lightweight, touch-first operating environment for Raspberry Pi devices.

Designed for:
- Raspberry Pi 3+
- SPI touch displays
- Kiosk systems
- Embedded devices

Current Version

0.0.1 Alpha

Status

🚧 Under Development

Goals

- Fast boot
- Touch-first interface
- Modular architecture
- Android-inspired UX
- Lightweight
- Themeable
- Open source
